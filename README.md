## About This Project

Event Flow is a simple task management system designed to help users manage tasks, monitor progress, and improve teamwork efficiency using GitHub collaboration.

The system demonstrates Agile methodology, GitHub collaboration, and version control in managing a team-based software development project.

## Agile Methodology

This project follows the **Agile Scrum methodology** using GitHub Issues, Branches, Pull Requests, and Project Board for collaboration and task tracking.

Scrum was selected because it supports teamwork, short development cycles, continuous feedback, and incremental improvement.
