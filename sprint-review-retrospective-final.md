### Lessons Learned 📚

| Lesson | Description |
|--------|-------------|
| Research first | Research new technologies before committing to them in a sprint |
| Clear roles help | Having defined Scrum roles made task allocation straightforward |
| Communication is key | Daily WhatsApp stand-ups kept everyone aligned throughout |
| Iterative works | Building features sprint by sprint made the project manageable |
| Firebase is powerful | Firebase made database integration simpler than expected |
| Test early | Testing features as they are built saves time fixing bugs later |
| GitHub improves | Version control collaboration improves significantly with practice |
| Velocity grows | Team velocity naturally increases as the team gains confidence |
